# AskToMike

A multi-agent system likely derived from or related to the Delegate project, featuring a frontend, backend, and automation agents.

## Structure

- **`ask-to-mike/`**: Frontend application (Next.js/React).
- **`backend-server/`**: Python-based real-time coordination server (LiveKit).
- **`my-stagehand-app/`**: Node.js automated browser agents.
